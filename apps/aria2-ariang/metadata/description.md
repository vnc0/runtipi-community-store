Aria2 is a lightweight multi-protocol & multi-source command-line download utility. It supports HTTP/HTTPS, FTP, SFTP, BitTorrent and Metalink.

## Features

- **Multi-protocol support**: Download files via HTTP/HTTPS, FTP, SFTP, BitTorrent, and Metalink
- **Modern web interface**: AriaNg provides a clean, responsive UI that works on desktop and mobile devices
- **Lightweight**: Minimal resource usage while maintaining high performance
- **Download management**: Pause, resume, and manage multiple downloads simultaneously
- **Torrent support**: Built-in BitTorrent client functionality
